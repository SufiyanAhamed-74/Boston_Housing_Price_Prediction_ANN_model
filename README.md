# 🏠 Boston Housing Price Prediction

## 📌 Project Overview

This project focuses on analyzing the famous Boston Housing dataset and building a Deep learning model to predict house prices based on different housing and environmental factors.

The project includes:

* Data Cleaning
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Data Visualization
* Regression Modeling
* Model Evaluation

---

# 🎯 Business Problem

Real estate prices are influenced by multiple factors such as crime rate, number of rooms, pollution level, tax rate, and location-related features.

The goal of this project is to analyze these factors and predict the median value of houses using Deep learning techniques.

---

# 📂 Dataset Information

Dataset: Boston Housing Dataset

Target Variable:

* `MEDV` → Median value of owner-occupied homes

Feature Columns:

* `CRIM` → Per capita crime rate by town
* `ZN` → Residential land zoned for large lots
* `INDUS` → Non-retail business acres per town
* `CHAS` → Charles River dummy variable
* `NOX` → Nitric oxide concentration
* `RM` → Average rooms per dwelling
* `AGE` → Proportion of old houses
* `DIS` → Distance to employment centers
* `RAD` → Highway accessibility index
* `TAX` → Property tax rate
* `PTRATIO` → Pupil-teacher ratio
* `B` → Black population index
* `LSTAT` → Lower status population percentage

Models were evaluated using:

---

# 🔍 Key Insights

* Houses with more rooms (`RM`) generally have higher prices.
* High crime rate (`CRIM`) negatively impacts house prices.
* Areas with high lower-status population percentage (`LSTAT`) tend to have lower house values.
* Pollution levels (`NOX`) also affect housing prices.
